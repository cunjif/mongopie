INSTALL AND TEST
======

Install pymongo
------

```
% sudo easy_install -U pymongo
```

Install
------

```
% cd mongopie
% sudo python setup.py install
```

Run example
-------

* Start mongod

* run example

```
% cd mongopie
% python mongopie_test.py
```

Please look at mongopie_test.py for detailed usages
